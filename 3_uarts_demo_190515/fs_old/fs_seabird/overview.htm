ADE-7491 Series Product Line


As the first ARM® Cortex™-M4 microcontrollers (MCUs) to be implemented on TI's proprietary 65-nm process technology, the Foxconn ADE-7491 Series using TI's C series MCU to provides customers an opportunity to increase their connectivity integration without sacrificing price, performance, or power consumption. This Series MCU integrates a floating-point core and high-performance analog while still providing low-power modes to offer a great set of functional options. The Foxconn ADE-7491 Series provides an excellent baseline for home, building, and industrial applications.
